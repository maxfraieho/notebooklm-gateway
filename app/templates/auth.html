Security Notice: This server has no authentication. Do not expose to public internet. For internal/testing use only.

How to get storage_state.json:

  1. Install notebooklm-py on your local machine:
    pip install notebooklm-py playwright
  2. Install Playwright browser:
    playwright install chromium
  3. Run login command (opens browser):
    notebooklm login
  4. Sign in with your Google account
  5. Find storage_state.json in current directory
  6. Upload the file below
