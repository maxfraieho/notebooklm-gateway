## Overview

exodus.pp.ua is a digital garden project that serves as a knowledge management platform. It is built on the Garden Bloom architecture and implements a sophisticated knowledge graph system.

## Architecture

### Knowledge Graph
The project implements a knowledge graph architecture with:
- Graph contract defining relationships
- Semantic edge types (proposed enhancement by [[people/olena]])
- Wiki-link based connections between knowledge entities
- Invariants defined in [[інваріанти_графу_знань]]

### Technical Foundation
- Based on Garden Bloom execution platform
- Storage-centric architecture with MinIO
- Agent-based system for knowledge integrity
- Proposal-based mutation model
