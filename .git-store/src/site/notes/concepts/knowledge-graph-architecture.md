## Overview

Knowledge graph architecture for exodus.pp.ua defines how knowledge entities are connected and related within the digital garden system.

## Components

### Graph Contract
The graph contract defines the rules and structure for how nodes (knowledge entities) connect to each other in the knowledge graph.

### Semantic Edge Types
Proposed enhancement by [[people/olena]] to add semantic edge types to the graph contract. This would provide more meaningful and typed relationships between knowledge entities beyond simple wiki-links.

## Implementation Context

The knowledge graph architecture is implemented within the Garden Bloom framework, which includes:
- [[інваріанти_графу_знань]] - Knowledge graph invariants
- Wiki-link based connections
- Structural validation rules
- Agent-based integrity checking
